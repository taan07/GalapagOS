## Forbidden

No git tags/checkpoints, no workers, no monitoring, no new SQLite tables, no
record types beyond the 8, no transcript dumps into records (doctrine only).

## Exit criterion

User tells the manager the project goal in chat → a committed
`docs/galapagos/goals/*.md` `active_goal` record exists; force a fresh session
(delete sdk session id or simulate resume failure) → the manager recalls the
goal via re-brief.
