## Forbidden

No monitor interval loop, no confidence engine, no auto-triage, no evidence
runner, no editing of the prior prototype repo, no shared files between lanes.

## Exit criterion

Via chat, the user spawns a worker on a scoped task in its own worktree,
watches it stream in `/workers`, steers it mid-run; a deliberate out-of-lane
edit made via Bash is caught at worker stop and raises a `lane_violation`
attention item.
